# type

Minimal typing trainer for QWERTY, Dvorak, Colemak-DH, and "Baremak".

"Baremak" keeps Colemak-DH letters and adds my custom symbol layer.

```text
base
q w f p b   j l u y ;
a r s t g   m n e i o '
z x c d v   k h , . /
```

```text
right alt
! @ # < /   \ > & $ |
[ { = ( +   - ) ~ } ] `
% ^ * _ ?   : ; ' "
```
